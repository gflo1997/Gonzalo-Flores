Here are some of my projects

  • Ultraverse - NFT World

    Html, CSS, JavaScript XML, React, React-Slick, AOS

    NFT website created as a project for "Frontend Simplified" bootcamp. Slick Carousel created using React-Slick. API was fetched using Axios. A skeleton loading stage was added for slower internet connections. Animations were added using AOS library. Finished project was deployed to Vercel.

  • "Internet Flicks" (Netflix Clone)

    HTML, CSS, JavaScript XML, React

    Developed a fully responsive movie app using React, incorporating features such as a filter system and pagination. Utilized Axios to process API requests and retrieve data from a cloud server, dynamically representing the data through skeleton loading states and dynamic routing.

  • Google Clone

    Html, CSS, JavaScript XML, React, Firebase

    Developed a search engine using Google's API that allows users to search for and sort news, images, and videos. Integrated the API and implemented search functionality using HTML, CSS, and JavaScript. Created a visually appealing and user-friendly interface that emulates the design of the real Google search engine. The resulting project demonstrated my ability to work with APIs and create functional and aesthetically pleasing web applications.

  • Twitter Clone Project

    Html, React, Next.js, Sanity CMS, Typescript, SSR, Tailwind & NextAuth

    Developed a social media app using React that allows users to sign in with their Twitter account, post tweets, comment, share, and message other users. Utilized the Twitter API to authenticate users and retrieve and post data. Created a visually appealing and user-friendly interface that emulates the design of the real Twitter app. The resulting project demonstrated my ability to work with APIs and create functional and aesthetically pleasing web applications.

  • Hulu Clone Project

    Html, React, Next.js, and Tailwind CSS

    Developed a fully responsive clone of the popular streaming app Hulu using Next.js and integrating multiple APIs, including the TMDb and OMDb APIs. Created a user-friendly interface with features such as an animated carousel and dynamic routing. Utilized Axios to process API requests and retrieve data, dynamically representing the data through skeleton loading states. The resulting app provided a seamless and enjoyable user experience for streaming enthusiasts.

  • YouTube Clone Project

    Html, CSS, JavaScript, React, and Material UI

    Developed a web application for searching and viewing YouTube videos using HTML, CSS, JavaScript, React, and the YouTube API. Led the project as the lead developer, designing the user interface and implementing search, video playback, and API integration. Created a visually appealing and user-friendly application that received positive feedback.

  • Library Project

    Html, CSS, JavaScript XML, React, JSON server

    Developed a personal library management system using React that allows users to browse, search, and 'buy' books. Implemented features such as filtering, pagination, and CRUD (create, read, update, delete) operations for books and users. Utilized a JSON server as a mock API to simulate data storage and retrieval. The resulting app demonstrated my ability to create functional and user-friendly web applications using React.
